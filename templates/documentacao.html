Documentação da Linguagem Linex

Bem-vindo à documentação oficial da linguagem de programação Linex. Criada para ser simples e poderosa, Linex permite que você execute comandos sequenciais e lógicos de forma clara e objetiva. Esta documentação é o seu guia completo para dominar a linguagem.

1. Introdução

O que é Linex?
Linex é uma linguagem interpretada, ideal para automação de tarefas simples, processamento de dados e scripts. Sua filosofia de design prioriza a legibilidade e a sintaxe intuitiva, tornando-a perfeita para iniciantes em programação.

---

2. Sintaxe Básica

Todo programa Linex deve começar com o comando de inicialização. Os comandos são executados em ordem, linha por linha.

linex init project

# Seu codigo comeca aqui...
# Cada comando em uma nova linha.

Comentários

Use o caractere # para adicionar comentários e anotações. O interpretador ignora qualquer coisa após ele.

# Este e um comentario sobre a proxima linha.
linex print "Iniciando o script..."

3. Comandos Principais

linex print "msg" | linex print var

Exibe uma mensagem ou o valor de uma variável. O operador + pode ser usado para concatenar texto e variáveis.

linex print "Ola, mundo!"
linex print "O resultado e: " + resultado

var nome = valor

Declara ou atualiza uma variável. O valor pode ser uma string (entre aspas), um número, ou uma expressão de cálculo.

var nome = "Miguel"
var idade = 30
var ano_nascimento = calc 2024 - idade

calc expressao

Realiza uma operação matemática e retorna o resultado. É frequentemente usado em conjunto com o comando var. A expressão pode conter variáveis e funções de bibliotecas padrão.

calc 10 * 5
var valor_final = calc (100 + 50) / 2

input variavel

Lê uma linha de entrada e a armazena na variável especificada.

input nome_do_usuario

request get "url" to var_name

Faz uma requisição HTTP GET para uma URL e salva o conteúdo da resposta na variável especificada.

request get "https://api.github.com/users/octocat" to dados_json

json load var_name_string to var_name_json

Converte uma string JSON armazenada em uma variável para um objeto Linex, permitindo o acesso a seus dados internos.

json load dados_json to dados_obj
linex print dados_obj.login # Acessando um campo do objeto
linex print dados_obj.id

sleep segundos

Pausa a execução do script por um número de segundos especificado.

linex print "Aguardando 2 segundos..."
sleep 2

if condicao begin ... end if

Executa um bloco de código se a condição for verdadeira. Também suporta else para um bloco alternativo.

if idade > 18 begin
    linex print "Voce e maior de idade."
else begin
    linex print "Voce e menor de idade."
end if

loop numero begin ... end loop

Executa um bloco de código um número fixo de vezes.

loop 3 begin
    linex print "Loop em execucao."
end loop

func nome begin ... end func

Define uma função, um bloco de código reutilizável.

func saudacao begin
    linex print "Ola, do interior da funcao!"
end func

call nome

Executa o código de uma função.

call saudacao

save "arquivo" | load "arquivo"

save: Salva todas as variáveis em um arquivo JSON.
load: Carrega variáveis de um arquivo JSON.

save "dados_do_jogo"
load "dados_do_jogo"

4. Bibliotecas Padrão

A linguagem Linex possui acesso a diversas funções e módulos de bibliotecas padrão do Python para tarefas comuns. Você pode usá-los dentro de expressões e comandos como calc e var.

math

Acesse funções matemáticas como math.sqrt() (raiz quadrada) e math.pi (o valor de PI).

var area = calc math.pi * 5 * 5
linex print "A raiz quadrada de 16 e: " + math.sqrt(16)

random

Gere números aleatórios com random.randint().

var numero_aleatorio = calc random.randint(1, 10)
linex print "Seu numero da sorte e: " + numero_aleatorio

datetime

Obtenha a data e hora atuais usando datetime.now().

var agora = datetime.now()
linex print "Data e hora atuais: " + agora

Counter

Use a estrutura de dados Counter para contar elementos em uma coleção.

var lista_frutas = ["maca", "banana", "maca", "laranja"]
var contagem = Counter(lista_frutas)
linex print "Contagem de frutas: " + contagem

5. Exemplo de Código Completo

Este script demonstra o uso de vários comandos para criar um pequeno programa interativo.

linex init project

# Boas-vindas
linex print "Bem-vindo ao Linex!"
linex print "Versão 1.2"
sleep 1

# Entrada e calculo com bibliotecas
input nome_usuario
var idade = calc 2025 - 1990 # Exemplo de calc

# Requisição web
request get "https://jsonplaceholder.typicode.com/posts/1" to post_data
json load post_data to post_obj

# Exibindo resultado e usando condicional
linex print "Ola, " + nome_usuario + "!"
linex print "O titulo do post e: " + post_obj.title
linex print "O ID do usuario e: " + post_obj.userId

# Chamando função e loop
func loop_message begin
    linex print "Executando loop..."
end func

loop 3 begin
    call loop_message
end loop

# Fim do programa
linex print "Fim da execucao."
